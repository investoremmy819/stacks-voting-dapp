# Stacks Voting DApp

A simple on-chain voting application built on Stacks using Clarity and the Stacks SDK.

## Features
- One vote per wallet
- On-chain vote counting
- Wallet connection with @stacks/connect
- Contract calls with @stacks/transactions

## Setup
1. Install Hiro Stacks Wallet
2. Switch to Stacks Testnet
3. Deploy `contracts/voting.clar`
4. Verify the contract on Hiro Explorer
5. Update `CONTRACT_ADDRESS` in `frontend/app.js`
6. Open `frontend/index.html` and connect wallet
